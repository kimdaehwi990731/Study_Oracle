# **SELECT**
## **00. 구성**
```
SELECT 데이터 컬럼1, 데이터 컬럼2 ..
FROM 테이블명;
```
- 조회쿼리는 데이터의 추가, 삭제, 수정이 일어나지 않음 → 커밋이나 롤백의 사용이 불필요함
- 컬럼명 대신 *을 활용하는 경우 해당 테이블에 존재하는 모든 컬럼의 데이터를 조회

## **01. 기능**
### **01-01. 별칭**
```
SELECT 데이터 컬럼1 (AS) "별칭" ...
FROM 테이블명 (AS) "별칭";
```
- AS 생략 가능

### **01-02. NVL**
```
SELECT NVL(데이터 컬럼. '대체 문자')
FROM 테이블명;
```
